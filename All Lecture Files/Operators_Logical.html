Logical Operators :

Logical operators are used to combine conditions or invert results.
They return a Boolean value â†’ true or false. Often used inside if statements, loops, and conditions.


AND (&&) :

Returns the first falsy operand. If all values are truthy, it returns the value of the last operand (Returns true only if both conditions are true. If any one is false, the result is false.)

You will get output in true and false( 0 = false 1 = true).

Or (||):

Returns true if at least one condition is true. Only false if both conditions are false.

Not (!):

Reverses the condition. true becomes false, false becomes true.
