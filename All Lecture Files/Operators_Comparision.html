Comparision Operator : Used to compare two values, returns true or false.

Greater Than (>) :Checks if left value is larger.

Lesser Than (<) : Checks if left value is smaller.

Greater than or equal to (>=) : True if left value is larger or equal.

Less than or equal to (<=) : True if left value is smaller or equal.

Equal to (loose comparison = =): Checks if two values are the same (ignores type).

Equal to (loose comparison = =)

Strict equal to (type + value = = =) : Checks if values and types are the same.

Strict equal to (type + value = = =)

Not equal to (=!)

Not equal to (=!) : Checks if values are different (ignores type).
